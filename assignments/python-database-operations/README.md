# 📘 Assignment: Python Database Operations

## 🎯 Objective

Learn to connect to SQLite databases, perform CRUD operations, and manage data persistence with Python's sqlite3 module.

## 📝 Tasks

### 🛠️ Create Database and Table

#### Description
Set up a SQLite database and create a table for storing student records.

#### Requirements
Completed program should:

- Create a new SQLite database file
- Define a table named 'students' with columns: id (INTEGER PRIMARY KEY), name (TEXT), grade (INTEGER)


### 🛠️ Insert and Read Data

#### Description
Add student records to the database and retrieve them to display.

#### Requirements
Completed program should:

- Insert at least 3 student records into the table
- Query and print all records from the database


### 🛠️ Update and Delete Data

#### Description
Modify existing records and remove records from the database.

#### Requirements
Completed program should:

- Update the grade of one student
- Delete one student record
- Display the final state of the database
