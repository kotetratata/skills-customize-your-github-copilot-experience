
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a Hangman game using Python strings, loops, conditionals, and user input. This assignment helps students practice string manipulation, control flow, and random selection while creating an interactive guessing game.

## 📝 Tasks

### 🛠️ Build the Hangman game

#### Description

Create the classic word-guessing game. Your program should choose a random word from a predefined list, allow the player to guess letters, display the current progress, and end when the word is guessed or when the player runs out of attempts.

#### Requirements

Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses and show current progress in `_ _ _` format
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display a win message when the word is guessed and a lose message otherwise
